How to Use

Basic Operations:
- Click buttons or use keyboard: `+`, `-`, `×`, `÷`
- Press `=` or `Enter` to compute
- Decimal: `.`
  
Special Functions:
- `AC`: Clear current input (or `Esc` key)
- Double-click `AC`: Clears memory
- `DEL`: Delete last digit (`Backspace` key)

Keyboard Shortcuts:
| Key | Action         |
|-----|----------------|
| 0–9 | Numbers        |
| .   | Decimal point  |
| + - * / | Operators (`*` = ×, `/` = ÷) |
| Enter / = | Calculate |
| Esc | Clear (AC)     |
| Backspace | Delete last digit |
